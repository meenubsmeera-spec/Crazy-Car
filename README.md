🚗 Crazy Car Game

A fun and beginner-friendly car animation game developed using HTML, CSS, and JavaScript. The car moves automatically around a track while a timer records the duration. Interactive elements such as trees, traffic signals, and pedestrians make the game visually engaging. Hovering over the car pauses both the movement and the timer, creating an interactive user experience.

Features:
• Automatic car movement around the road
• Real-time timer display
• Interactive hover effects
• Attractive game environment with emojis
• Built entirely with HTML, CSS, and JavaScript

This project was created to practice front-end web development concepts including animations, DOM manipulation, event handling, and timers.
